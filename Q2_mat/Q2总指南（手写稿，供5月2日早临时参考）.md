# 3.定量求分段点
## 3.1 尝试性探究线性速度

首先可视化目测速度为线性，故做匀加速直线运动拟合进行**尝试性探究**。

MATLAB signal processing toolbox自带变点检测函数findchangepts，当统计方法参数设置为linear时，即可自动按全局线性拟合残差最小化作为目标进行分段。得到速度的分段函数后，反推出位移随时间变化的规律，再与清洗后的原始“位移-时间”数据进行残差分析。最终结果为：

- 阶段Ⅰ：0.0 ~ 79570.0 min。0.000287 + 0.000000 * t (加速度 = 0.000000 mm/min²)。位移 RMSE = 0.640815 mm。

- 阶段Ⅱ：79580.0 ~ 94970.0 min。v(t) = -0.211275 + 0.000003 * t (加速度 = 0.000003 mm/min²)。位移 RMSE = 0.419544 mm。

- 阶段Ⅲ：94980.0 ~ 96960.0 min。v(t) = -3.112779 + 0.000033 * t (加速度 = 0.000033 mm/min²)。位移 RMSE = 0.250823 mm。

- 阶段Ⅳ：96970.0 ~ 99990.0 min。v(t) = 0.098941 + 0.000000 * t (加速度 = 0.000000 mm/min²)。位移 RMSE = 0.423900 mm。终末期。此时滑坡已经发生，脱离前失稳阶段预警规律的研究范畴，无研究意义。


结果：阶段Ⅱ与阶段Ⅲ残差小，拟合结果好。阶段Ⅰ残差大，说明这一部分速度用单段线性函数刻画并不准确。


> [!tip] 分析原因：
> - **早期阶段（缓慢匀速阶段/加速）**：速度波动以噪声为主，速度的斜率难以辨别。`'linear'`就无法正确识别出该部分的分段点。
> - **后期阶段（加速/快速阶段）**：速度开始呈现趋势性增长，`'linear'` 统计量对**斜率变化**比 `'rms'` 更敏感，能更准确地找到加速→快速的转折点。

因此针对前期再次展开分段研究，找出“加速形变期”的具体开始时间节点。

## 3.2 优化调整初期分段

`'rms'` 统计量对**均值变化**非常敏感，更适合捕捉此时从“稳定均值”到“均值开始升高”的转变。出于3.1分析原因中提出的明确物理动机，我们混合使用两种统计量。
在平稳段用“均值突变”判据，在增长段用“趋势突变”判据，这样更符合边坡形变的物理过程。

将变点检测函数findchangepts的统计方法参数设置为rms，对阶段Ⅰ检测变点，再一次分段，最终得到了“缓慢匀速形变->加速形变”的分界点，为23100.0 min。

- 子阶段 1（t = 0.0 ~ 23100.0 min）: 位移 RMSE=0.409126 mm, MAE=0.370909 mm, Max=0.835674 mm ，即缓慢匀速形变期

- 子阶段 2（t = 23110.0 ~ 79570.0 min）: 位移 RMSE=0.535956 mm, MAE=0.433175 mm, Max=1.160338 mm，即加速形变期

成功解决了单个线性速度分段位移均方根误差过高的情况。

综上所述：
- 缓慢匀速形变：0.0 ~ 23100.0 min
- 加速形变：23110.0 ~ 79570.0 min
- 快速形变Ⅰ：79580.0 ~ 94970.0 min (加速度 = 0.000003 mm/min²)
- 快速形变Ⅱ：94980.0 ~ 96960.0 min (加速度 = 0.000033 mm/min²)
