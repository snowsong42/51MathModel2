$$
A = \alpha B + \beta,\quad \alpha = 1.1346,\;\beta = 0.5630\ \mathrm{mm}

$$
A_{\mathrm{corr}} = \frac{A - \beta}{\alpha} = \frac{A - 0.5630}{1.1346}

## 4. 关键公式（可直接复制到 LaTeX）

**MAD 定义**：
```latex
\mathrm{MAD}(\mathbf{r}) = \mathrm{median}\bigl(|r_i - \mathrm{median}(\mathbf{r})|\bigr)
```

**线性回归模型**：
```latex
A_i = \alpha B_i + \beta + \varepsilon_i,\quad \varepsilon_i \sim \mathrm{WN}(0, \sigma^2)
```

**校正变换**：
```latex
A_{\mathrm{corr},i} = \frac{A_i - \hat{\beta}}{\hat{\alpha}}
```

**MAE 与 RMSE**：
```latex
\mathrm{MAE} = \frac{1}{n}\sum_{i=1}^n |B_i - A_{\mathrm{corr},i}|,\quad
\mathrm{RMSE} = \sqrt{\frac{1}{n}\sum_{i=1}^n (B_i - A_{\mathrm{corr},i})^2}
```

**前向滚动窗口交叉验证**：第 k 折训练集为前 k 块，测试集为第 k+1 块。

## 5. 概念表述建议（论文中直接用）

> **MAD（绝对中位差）**：一种稳健的离散度度量，相比标准差对极端值不敏感。对于近似正态数据，MAD ≈ 0.6745σ。

> **互相关函数**：衡量两个信号在不同时间偏移下的相似程度。峰值位置对应最优时间延迟，峰值接近 1 表明高度线性相关。

> **前向滚动窗口交叉验证**：严格保持时间因果的验证策略——仅用历史数据训练、未来数据测试，避免"未来信息泄露"导致的乐观估计。

> **Durbin-Watson 检验**：检验残差是否存在一阶自相关。统计量接近 2 表示无自相关，接近 0 或 4 分别表示正/负自相关。

> **Ljung-Box Q 检验**：检验残差在多个滞后阶数上是否整体为白噪声。p > 0.05 则不拒绝白噪声假设。
