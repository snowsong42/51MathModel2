### 5.4.2 问题四模型建立

基于 5.4.1 节构造的特征体系，本文采用**分阶段 LightGBM 回归模型**对表面位移进行预测。考虑到边坡形变具有缓慢变形、加速变形、快速变形三个物理阶段，且各阶段中外界扰动（如爆破、降雨）对位移的影响机制显著不同，因此为每个阶段单独训练一个回归模型。

**模型结构**

记 tt 时刻的特征向量为 x(t)∈Rnx(t)∈Rn，其中 nn 为特征工程后的特征总数。将时间序列按阶段划分为三段：

- 缓慢变形阶段：t∈[0,b1)t∈[0,b1​)，对应 Phase 0
    
- 加速变形阶段：t∈[b1,b2)t∈[b1​,b2​)，对应 Phase 1
    
- 快速变形阶段：t∈[b2,T]t∈[b2​,T]，对应 Phase 2
    

其中 b1,b2b1​,b2​ 为表 28 给出的拐点索引，TT 为序列总长度。

对每个阶段 p∈{0,1,2}p∈{0,1,2}，训练一个 LightGBM 回归模型 MpMp​，其预测目标为**单步位移增量** ΔDisp(t)ΔDisp(t)（即相邻时刻位移的差分）：

ΔDisp(t)=Mp(x(t))ΔDisp(t)=Mp​(x(t))

选择 ΔDispΔDisp 作为直接预测目标，是因为位移增量（即速度的线性缩放）在物理上更能反映边坡外界扰动的瞬时响应，且通过累积可自然得到位移序列。

**训练策略**

为增强模型鲁棒性并减少随机性，对每个阶段采用**多种子平均**策略：

1. 使用 3 个不同随机种子（42, 43, 44）分别训练 3 个 LightGBM 模型；
    
2. 预测时取 3 个模型输出的均值作为最终预测值。
    

**位移重构**

由于模型输出的是单步位移增量，实际位移需通过累积求和得到：

Disp(t)=Disp(0)+∑k=1tΔDisp(k)Disp(t)=Disp(0)+k=1∑t​ΔDisp(k)

其中初始位移 Disp(0)Disp(0) 取该阶段的起始位移实测值。对于实验集预测，初始位移取训练集最后一个时刻的位移值，以保证位移曲线的连续性。

**LightGBM 超参数设置**


正则化项的引入可有效防止模型在噪声较强的快速变形阶段过拟合，保证对实验集的泛化能力。
