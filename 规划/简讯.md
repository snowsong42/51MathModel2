# 五一数学建模竞赛综述

## 一、竞赛概况

五一数学建模竞赛是由江苏省工业与应用数学学会、徐州市工业与应用数学学会、中国矿业大学主办，中国矿业大学学生社团——大学生数学建模协会承办的源于江苏、面向全国、辐射国际的学术科技竞赛活动。竞赛于 2004 年由中国矿业大学数学系大学生发起，旨在调动学生学习数学的积极性，在面对实际问题寻求解决方案过程中，提高学生建立数学模型和运用计算机技术的综合能力。

自 2004 年首次办赛至今，五一数学建模竞赛已经连续成功举办 22 年，累计吸引海内外 20 余万学生参赛，办赛规模不断壮大、竞赛质量不断提升。2025 年，竞赛成功入选江苏省普通高校本专科生学科竞赛省级赛事目录。目前年参赛规模超万人，覆盖全国 31 个省、自治区和直辖市以及多个国家。

### 竞赛基本信息

| 项目 | 内容 |

### 奖项设置

竞赛按组别分别设一等奖、二等奖、三等奖和成功参赛奖。各组一、二、三等奖数量分别不超过所在组别报名队数的 5%、15%、25%；凡成功提交有效论文的参赛队均可获得成功参赛奖。以 2022 年（第十九届）数据为例，共有 6648 支代表队报名参赛，评选出一等奖 291 篇（获奖比例 4.38%），二等奖 877 篇（13.19%），三等奖 1457 篇（21.92%），总获奖比例约 39.49%。


## 二、赛题总览

五一数学建模竞赛的赛题均来源于工程技术、经济管理、社会生活等领域中的实际问题，经过适当简化加工而成。没有事先设定的标准答案，留有充分余地供参赛者发挥其聪明才智。竞赛评奖以**假设的合理性、建模的创造性、结果的正确性和文字表述的清晰程度**为主要标准。

自 2004 年创办以来，组委会编写了五一数学建模竞赛历届赛题合集，收录了覆盖工程技术、经济管理和社会生活三大领域的 60 道赛题。题目具有较大灵活性，不要求参赛者预先掌握系统的专门知识，只需学习过普通高等数学相关课程即可参赛。

各题目有固定的核心考察领域和能力侧重，选题直接匹配团队的知识储备和技能优势即可，以下是**各题目的核心定位、考察方向、典型题型**的详细介绍，也是历年赛题的固定规律，适配备赛和选题参考。

### 五一数学建模 A/B/C 三题概览

> **注意**：与美赛不同，五一数学建模竞赛的 A/B/C 三题**不按“连续/离散/数据”严格划分**，而是以实际应用场景为导向命题。以下分类基于对历年赛题的统计分析和社区经验总结，作为选题参考。

### A 题：优化与工程决策类问题

1. **核心领域**：主要为工业工程、生产制造、物理过程、路径规划等需要**精确优化求解**的实际问题。
2. **考察重点**：优化理论与方法（线性规划、非线性规划、整数规划、多目标规划）、图论（最短路径、路径规划）、搜索算法（启发式搜索、遗传算法）、几何建模与物理建模。
3. **典型赛题示例**：
   - 工程优化：钢板最优切割路径问题（2024）、木板最优切割方案（2019）、工件的安装与排序问题（2005）；
   - 路径规划：不确定性条件下的最优路径问题（2015）、徐州潘安湖风景区游览路线设计（2018）、公交车排班问题（2017）；
   - 物理建模：无人机定点投放问题（2023）、让标枪飞（2019）、降落伞的选择（2004）。
4. **适配团队**：擅长优化算法和图论，能熟练使用 Lingo/Gurobi/Python 优化库，具备较强数学建模功底和算法实现能力的团队。

### B 题：综合分析与预测评价类问题

1. **核心领域**：经济管理、社会问题、政策分析等**需综合运用多种建模方法**的开放性问题，是五一赛中对逻辑论证和综合分析能力要求最高的题目。
2. **考察重点**：统计分析与预测（回归分析、时间序列预测、神经网络预测）、综合评价方法（AHP 层次分析法、TOPSIS、模糊综合评价）、系统建模与仿真、数据分析与处理。部分赛题涉及网络流、排队论、微分方程等多方法融合。
3. **典型赛题示例**：
   - 经济管理：商业银行人民币贷款规模分配及盈利问题（2018）、能源总量控制下的城市工业企业协调发展问题（2016）、科创板拟上市企业估值（2019）；
   - 社会问题：幸福感的评价与量化模型（2011）、“二孩政策”问题（2016）、自媒体时代的消息传播问题（2017）；
   - 工程综合：消防救援问题（2021）、快递需求分析问题（2023）。
4. **适配团队**：综合能力强的团队，擅长多种建模方法的灵活运用和融合，具备较强的数据分析和逻辑论证能力。

### C 题：数据科学与应用场景类问题

1. **核心领域**：以数据驱动的实际问题为核心，背景多涉及能源、环境、公共安全、“双碳”目标等社会关注热点，是五一赛中最贴近**实际数据应用**的题目。
2. **考察重点**：数据处理与特征工程、机器学习（分类、回归、聚类、集成学习）、信号处理与分析、统计建模与因果推断、预测建模。部分题目需收集外部数据，对数据获取和处理能力要求高。
3. **典型赛题示例**：
   - 能源与安全：煤矿深部开采冲击地压危险预测（2024）、碳排放约束下的江苏省煤炭消费预测（2012）、能源总量控制问题（2014）；
   - 环境与可持续：“双碳”目标下低碳建筑研究（2023）、空气污染问题研究（2015）、生态文明建设评价问题（2015）；
   - 数据预测与异常检测：数据驱动的异常检测与预警问题（2021）、煤炭价格预测问题（2020）。
4. **适配团队**：擅长 Python 数据处理与机器学习，掌握 Scikit-learn/TensorFlow 等工具，具备特征工程和模型调优能力的团队（备赛核心是数据建模和代码实现）。

### 三题差异与选题建议

| 题目 | 核心特点 | 核心能力要求 | 难度参考 |
|------|----------|-------------|---------|
| A 题 | 优化求解导向，目标明确、有合理结果范围 | 优化算法、图论、启发式搜索、几何数据解析 | 中等（方法明确但求解难度大） |
| B 题 | 综合分析导向，开放度高，多方法融合 | 统计预测、综合评价、系统建模、逻辑论证 | 最高（方法不明确，需多角度建模） |
| C 题 | 数据驱动导向，贴近实际热点问题 | 机器学习、数据处理、统计分析、特征工程 | 中到高（方法明确但数据细节坑多） |

**选题核心原则**（3 人团队适配）：

1. **优先选团队知识储备匹配的题目**，而非“看起来简单”的题目。A 题看似方法直接，但在优化求解上可能遇到算法效率瓶颈；C 题看似数据驱动易上手，但特征工程和数据预处理的细节往往影响模型质量。

2. **若团队算法和编程能力强**（Python/Gurobi 等），优先选 **A 题**（优化类问题方法明确）；**若数据分析与机器学习能力强**，优先选 **C 题**；**若综合能力强、擅长多角度论述和分析**，可选 **B 题**（开放度高，发挥空间大）。

3. 五一赛所有题目均来源于工程技术、经济管理和社会生活中的实际问题，论文中需明确体现**模型的实际应用价值**。

4. 五一赛赛题与时俱进，2020 年以来能源安全、“双碳”战略等热点成为命题重要来源，《“双碳”目标下低碳建筑研究》（2023）、《煤矿深部开采冲击地压危险预测》（2024）等均体现了这一趋势。


## 三、备赛工具与方法清单

按**A 题（优化与工程决策）、B 题（综合分析与预测评价）、C 题（数据科学与应用场景）**分题整理，每个题目明确**必备数学方法**（核心考察，优先掌握）、**核心编程工具**（代码实现/计算，五一赛主流 Python/Matlab/Lingo）、**专用建模软件**（辅助建模/仿真/求解），所有工具均贴合五一赛实战需求，同时标注**高频用法**，适配备赛重点突破。

### A 题：优化与工程决策类

#### 必备数学方法
1. 优化理论与方法（线性规划、整数规划、0-1 规划、混合整数规划、非线性规划——核心）
2. 多目标规划（Pareto 最优、目标规划法、加权法）
3. 图论（最短路径 Dijkstra/Floyd、最小生成树、TSP 问题）
4. 启发式搜索算法（遗传算法 GA、模拟退火、粒子群优化 PSO）
5. 几何建模与物理建模（空间坐标计算、受力分析、运动轨迹建模）
6. 动态规划（多阶段决策问题的优化求解）

#### 核心编程工具
- **Matlab**（主力）：Optimization Toolbox（fmincon/linprog/intlinprog 求解各类优化问题）、Global Optimization Toolbox（遗传算法 ga/模拟退火 simulannealbnd）、Mapping Toolbox（路径规划）、可视化出图
- **Python**（主力）：PuLP/Gurobipy/CPLEX（求解整数/混合整数规划）、SciPy.optimize（非线性规划 minimize）、NetworkX（图论建模与路径计算）、DEAP（遗传算法框架）、NumPy（矩阵/几何计算）
- **通用**：Latex（论文排版与公式书写，五一赛论文必备）

#### 专用建模软件
1. **Lingo/Lindo**（核心）：轻量化优化求解器，无需大量代码，快速求解线性/整数/非线性规划问题。五一赛 A 题入门首选，上手极快，适合编程基础薄弱的团队
2. **Gurobi/CPLEX**（进阶）：高性能优化求解器，支持复杂多约束优化，可与 Python/Matlab 联动（适合求解大规模复杂赛题）
3. **AutoCAD**（辅助）：部分工程类赛题（如切割方案、空间布局）的几何图形绘制与解析

### B 题：综合分析与预测评价类

#### 必备数学方法
1. 统计预测方法（回归分析、时间序列 ARIMA/指数平滑、灰色预测 GM(1,1)、神经网络预测）
2. 综合评价方法（AHP 层次分析法、TOPSIS 法、模糊综合评价、熵权法——权重计算与方案排序核心）
3. 统计分析（描述性统计、假设检验、相关分析、因子分析）
4. 系统建模（系统动力学、排队论、微分方程建模——部分赛题）
5. 博弈论（公共政策分析、多方决策问题）
6. 网络分析与传播模型（信息传播、社交网络分析部分赛题）

#### 核心编程工具
- **Matlab**（主力）：统计工具箱（回归分析 regress、假设检验）、曲线拟合工具箱（fitnlm/cftool）、神经网络工具箱（nftool 快速预测建模）、Simulink（系统动力学仿真）
- **Python**（辅助）：Scikit-learn（回归预测 RandomForest/SVR）、StatsModels（ARIMA 时间序列分析）、NumPy（AHP/TOPSIS 计算和矩阵运算）、Pandas（数据分析与处理）、Matplotlib/Seaborn（可视化出图）
- **通用**：Latex（论文排版，B 题论证过程复杂，公式简洁呈现极为重要）、Visio/亿图图示（流程绘图和逻辑框架图）

#### 专用建模软件
1. **SPSS**（核心）：轻量化统计分析工具，快速做回归分析、因子分析、假设检验。B 题可选首选，帮助编程基础薄弱团队加速统计分析
2. **YAAHP**（核心）：轻量化 AHP 层次分析法工具，无需编程，快速计算权重和一致性检验。五一赛 B 题最常用工具之一
3. **Vensim**（辅助）：系统动力学建模工具，可视化构建复杂反馈系统（适用于政策效果仿真、社会经济系统建模，部分赛题有需求）
4. **Excel**（高频辅助）：快速做简单统计、表格、权重计算，适合快速数据处理和方案对比

### C 题：数据科学与应用场景类

#### 必备数学方法
1. 数据处理与特征工程（数据清洗、缺失值处理、异常值检测、特征提取与降维 PCA）
2. 机器学习（分类：SVM/随机森林/XGBoost；回归：岭回归/Lasso/GBDT；聚类：KMeans/DBSCAN）
3. 深度学习（轻量级模型：BP 神经网络、卷积神经网络 CNN——部分复杂数据规律学习场景）
4. 信号处理与分析（傅里叶变换、小波变换、滤波处理——能源/安全类赛题常见）
5. 统计建模（相关分析、假设检验、回归诊断）
6. 预测建模（时间序列预测、多变量预测、集成预测方法）

#### 核心编程工具
- **Python**（绝对主力，五一赛 C 题首选）：
  - 数据处理：Pandas/NumPy（清洗、预处理、描述性统计）
  - 建模：Scikit-learn（传统 ML 算法：KMeans/SVM/随机森林/XGBoost）、StatsModels（统计回归/时间序列）
  - 信号处理：SciPy.signal（滤波、频域分析）
  - 可视化：Matplotlib/Seaborn/Plotly（静态/交互式可视化，论文出图）
  - 进阶：TensorFlow/PyTorch（轻量深度学习，如简单神经网络预测）
- **Matlab**（辅助）：统计与机器学习工具箱（Classification Learner 快速分类建模）、信号处理工具箱（时频分析）、神经网络工具箱
- **通用**：Latex（论文排版和数据表格呈现）

#### 专用建模软件
1. **Python 生态**（核心，无替代）：上述库直接满足全流程需求（五一赛 C 题纯 Python 即可完成从数据清洗到建模到可视化的所有工作）
2. **SPSS**（辅助）：轻量化统计分析，适合快速做描述性统计、回归分析、假设检验（适合编程基础薄弱的团队快速完成基础分析部分）
3. **Tableau/Power BI**（辅助）：交互式数据可视化，快速生成高质量图表，提升论文视觉效果
4. **Origin**（辅助）：专业数据绘图软件，适合高精度科学数据可视化（部分能源/物理类赛题出图需求）


### 通用补充（五一赛所有题目必备）

1. **Latex**：五一赛论文极其重要的排版工具，公式、图表、排版整体水平对评审结果有重要影响（推荐在 Overleaf 或本地使用标准模板）
2. **Office Word**（实际常用）：尽管 Latex 是建模圈推崇的工具，但五一赛对排版格式要求灵活，Word 也完全可以胜任论文撰写，关键在于内容质量
3. **Visio / 亿图图示**：建模流程图、系统框架图、网络/流程绘图、论文图表可视化必备
4. **知网 / 万方 / Web of Science**：文献检索。在撰写论文背景和模型比较时需要有文献支撑。五一赛赛题往往与研究热点相关，适当引用文献可提升论文学术性
5. **AI 辅助工具**（如 ChatGPT、文心一言）：在符合竞赛规则和学术诚信前提下，可辅助文献检索、代码 debug、英文翻译等，但核心建模和论文撰写应由团队独立完成
6. **常见作图工具**：Matplotlib（Python）/ Plotly（交互图）—— 论文出图首选，保证图表科学规范


## 四、竞赛章程与评审要点

### 论文要求

参赛者需根据题目要求，完成一篇包括**模型的假设、建立和求解、计算方法的设计和计算机实现、结果的分析和检验、模型的改进**等方面的论文。论文需在竞赛截止时间前通过官网竞赛系统提交。

### 评审标准

竞赛评奖以以下四个方面为主要标准：
1. **假设的合理性**
2. **建模的创造性**
3. **结果的正确性**
4. **文字表述的清晰程度**

### 参赛纪律

竞赛期间参赛队员可使用各种图书资料、计算机软件等，也可通过互联网查阅相关资料，但不得与参赛队员以外的任何人（包括在网上）进行讨论。对于存在抄袭、剽窃、造假等违背学术诚信行为的参赛队，组委会将取消该队参赛资格，成绩视为无效，并向所在高校通报，同时取消下一年度的参赛资格。


## 五、竞赛影响力与参赛价值

五一数学建模竞赛经过二十余年发展，已逐步成为具有一定影响力的课外学术科技活动和实践创新教育的品牌赛事。2016年起，竞赛连续四年被纳入“赢在徐州——中国徐州创新创业大赛”系列赛事。中科院院士李大潜曾为该赛事题词，中国教育电视台两次专题报道。

对于大学生而言，五一数学建模竞赛是一项非常值得参加的赛事，具有多重参赛价值：

**1. 备战国赛/美赛的“练兵场”。** 五一赛采用与全国大学生数学建模竞赛（高教社杯）高度一致的竞赛形式与评审标准——通讯竞赛、赛题发布后限时解答、提交完整建模论文，为备战国赛和美赛提供了最真实的实战模拟。同时，五一赛竞赛时间恰逢五一假期（5月1日—5月4日），距离9月国赛刚好约4个月，是非常适合作息节奏的时间节点，既可以检验上半学期备赛成果，又能通过实战发现团队短板、为暑期冲刺备赛提供明确方向。部分高校已将五一赛参赛经历作为国赛参赛资格审核条件之一。

**2. 锻炼综合能力、提升数学实践创新能力。** 竞赛题目来源于社会实际需求，参赛过程近似于“真刀真枪”的训练，模拟了毕业后解决实际问题的工作场景。通过参与竞赛，学生能够在短时间内极大锻炼以下核心能力：查阅文献与资料收集能力、数学建模能力、编程实现与计算机技术运用能力、科技论文写作能力以及团队合作精神和抗压能力，综合素质将得到全面跃升。

**3. 助力保研、奖学金评定与升学就业。** 五一数学建模竞赛已入选江苏省普通高校本专科生学科竞赛省级赛事目录，含金量持续提高，获奖证书可在保研加分、综合测评、奖学金评定、评奖评优中发挥重要作用。部分高校将五一赛获奖成绩计入本科生素质发展学分，并纳入学校第二课堂成绩单体系。此外，数学建模竞赛经历本身就是保研简历和求职简历中的重要学术竞赛亮点，能体现参赛者的创新能力和解决实际问题的水平。

**4. 参赛门槛低、获奖机会大。** 与国赛需通过校级选拔才能参赛不同，五一赛可直接报名参赛，面向所有在校学生开放。获奖比例较高（一等奖5%、二等奖15%、三等奖25%，总获奖可达约40%以上），只要认真参赛并提交有效论文，有很大机会获得成功参赛奖及以上奖项。

**历年赛题命题趋势**：
- **2004—2010 年（初创期）**：赛题以社会热点和日常生活中的数学问题为主，如房地产、就业、保险等，题目开放性强，数据来源多依赖参赛者自行收集或假设。
- **2011—2016 年（发展期）**：赛题逐步向更专业的工程技术和经济管理领域拓展，能源问题、环境问题、政策问题开始频繁出现，建模方法要求更加系统化。
- **2017—2024 年（成熟期）**：赛题更加贴近现实产业需求和国家战略，数据驱动类题目比重显著上升，机器学习、大数据分析方法成为必备技能。尤其 2020 年后，能源安全、“双碳”战略等热点问题多次成为命题核心，体现了竞赛立足国家需求、服务经济社会发展的导向。
